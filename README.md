# Earthbase Directory

The place to discover how to invest in nature as a human.

This is a curated directory of platforms, organisations, and tools that make it possible for individuals to invest in, protect, and restore nature. From land ownership and tree planting to carbon credits and biodiversity markets — we bring them all together in one place.
